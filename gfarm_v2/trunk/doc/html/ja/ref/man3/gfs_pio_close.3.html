gfs_pio_close

Name

gfs_pio_close -- Gfarmファイルシステムのファイルのクローズ

Synopsis

#include <gfarm/gfarm.h>

gfarm_error_t gfs_pio_close(GFS_File gfp);

DESCRIPTION

ファイル gfp をクローズします。

RETURN VALUES

GFARM_ERR_NO_ERROR

正常終了を示します。

GFARM_ERR_BAD_FILE_DESCRIPTOR

gfp で指定されるファイルディスクリプタが不正です。

GFARM_ERR_NO_MEMORY

メモリ不足のため処理を中止しました。

その他

上記以外のエラーが生じたことを示します。 gfarm_error_string(3) によりエラーの原因を文字列に変換することができます。

SEE ALSO

gfs_pio_create(3), gfs_pio_open(3)
