gfchown

Name

gfchown -- Gfarm ファイルの所有者，所有グループを変更する

Synopsis

gfchown [options] owner[:group] [path...]

gfchown [options] :group [path...]

DESCRIPTION

Gfarm ファイルの所有者，所有グループを変更します．

OPTIONS

-h

シンボリックリンクの指すファイル／ディレクトリではなく， シンボリック自身の所有者，所有グループを変更します．

-R

再帰的に所有者，所有グループを変更します．

-?

引数オプションを表示します．
