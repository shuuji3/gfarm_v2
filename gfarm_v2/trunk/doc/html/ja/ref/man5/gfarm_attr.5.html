gfarm_attr

Name

gfarm_attr -- Gfarm拡張属性

DESCRIPTION

Gfarmの各ファイルやディレクトリは， 拡張属性を複数持つことができます． 拡張属性とは，名前に対する値のペアで構成されます．

拡張属性の名前空間

"gfarm."で始まる名前の拡張属性はGfarmが特別に扱います．それ以外の拡張属 性はユーザが自由に名前を指定して値を格納できます．

"gfarm."で始まる名前の拡張属性は誰でも参照できます． ファイルまたはディレクトリの所有者，またはgfarmrootグループに 所属するユーザーだけが"gfarm."で始まる名前の拡張属性を作成または変更で きます． それ以外の拡張属性は，ファイルまたはディレクトリのアクセス権限によって 読み書きできます．

Gfarm 拡張属性

以下はGfarmのシステムが使用する拡張属性名です．

gfarm.ncopy

gfmd で自動でファイルを複製する際に目標とする複製の合計数を格納します．

gfarm.acl_access

ACL(アクセスコントロールリスト)を格納します．

gfarm.acl_default

デフォルトACLを格納します． ディレクトリだけがこの拡張属性を利用します．
