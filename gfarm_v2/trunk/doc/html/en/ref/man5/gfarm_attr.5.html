gfarm_attr

Name

gfarm_attr -- Gfarm Extended Attributes

DESCRIPTION

Files and directories in Gfarm filesystem can have Extended Attributes. Extended attributes are name:value pairs.

EXTENDED ATTRIBUTE NAMESPACES

The name of Extended Attribute which has "gfarm." prefix is special in Gfarm filesystem. Gfarm users can freely use the other names of Extended Attribute.

Everyone can get the Extended Attributes named "gfarm.*". Owner or gfarmroot group can modify the Extended Attributes named "gfarm.*". The others of Extended Attribute can be read/modified on the files or directories permission.

GFARM EXTENDED ATTRIBUTES

The following is a list of the Extended Attribute names used in Gfarm filesystem.

gfarm.ncopy

The number of file replicas to be created automatically in gfmd.

gfarm.acl_access

Access ACL (Access Control List).

gfarm.acl_default

Default ACL. Only directories can have/use this Extended Attribute.
