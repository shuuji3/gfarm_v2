gfperf-copy

Name

gfperf-copy -- Gfarm ファイル複製の性能測定を行う。

Synopsis

gfperf-copy -s, --src コピー元url -d, --dst コピー先url [-l, --filesize ファイルサイズ] [-b, --busize バッファサイズ] [-g, --gfsd gfsd名] [-m, --gfarm2fs gfarm2fsマウントポイント]

DESCRIPTION

コピー元urlからコピー先urlにファイルをコピーしそのコピー性能を測定します。 コピー元urlとコピー先urlはどちらか一方がgfarm url (gfarm:///で始まるもの) でなくてはなりません。そしてもう片方はfile url (file:///で始まるもの) でなくてはなりません。

OPTIONS

-s, --src コピー元url

コピー元urlを指定します。コピー元urlにgfarm urlを指定した場合、コピー先urlはfile urlでなくてはなりません。

-d, --dst コピー元url

コピー先urlを指定します。コピー先urlにgfarm urlを指定した場合、コピー元urlはfile urlでなくてはなりません。

-l, --filesize ファイルサイズ

使用するファイルサイズを指定します。 デフォルトは1Mです。

-b, --bufsize バッファサイズ

使用するバッファサイズを指定します。 デフォルトは4Kです。

-g, --gfsd gfsd名

ファイルを作成するgfsdを指定します。 本オプションを指定しなかった場合はgfarmが自動的に割当てたgfsdを使用します。

-m, --gfarm2fs gfarm2fsマウントポイント

gfarm2fsのマウントポイントを指定します。 gfarm urlが指定された方に対しgfarm2fs経由でのアクセスを行います。
