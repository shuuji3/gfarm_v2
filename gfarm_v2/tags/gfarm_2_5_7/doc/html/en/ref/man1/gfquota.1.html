gfquota

Name

gfquota -- display the current usage and limits

Synopsis

gfquota [options]

DESCRIPTION

gfquota displays your current usage and limits. An administrator in the gfarmadm group can specify a user or a group to display. If you are a general user, you can specify a group which you belong to.

The count of usage and the check of quota are enabled after executing gfedquota and gfquotacheck command. And, gfquota can display the current usage and limits.

OPTIONS

-P NAME

Specifies a path name to specify a metadata server instead of the root metadata server.

-q

Do not display the current usage and limits. Warning messages are displayed only if the current usage exceeds the limits.

-u NAME

Specifies a user name to display the current usage and limits.

-g NAME

Specifies a group name to display the current usage and limits.

-?

Displays a list of command options.
