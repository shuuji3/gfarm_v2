gfs_acl_delete_perm

Name

gfs_acl_delete_perm -- delete a permission from a Gfarm ACL permission set

Synopsis

#include <gfarm/gfarm.h>

gfarm_error_t gfs_acl_add_perm(gfarm_acl_permset_t permset_d, gfarm_acl_perm_t perm);

INPUT ARGUMENTS

permset_d

An ACL permission set.

perm

GFARM_ACL_READ or GFARM_ACL_WRITE or GFARM_ACL_EXECUTE.

RETURN VALUES

GFARM_ERR_NO_ERROR

The function terminated successfully.

Others

An error except the above occurred. The reason is shown by gfarm_error_string(3).
