gfexport

Name

gfexport -- print a file in the Gfarm file system on the standard output

Synopsis

gfexport [-h filesystem-node] Gfarm-URL

DESCRIPTION

gfexport outputs a Gfarm file specified by Gfarm-URL to the standard output.

OPTIONS

-h filesystem-node

explicitly specifies a filesystem node to be accessed.

-?

displays a list of command options.
