Name

gftest — Gfarm ファイルまたはディレクトリの情報を調べる

Synopsis

gftest options path

DESCRIPTION

指定したファイルあるいはディレクトリの情報を調べます．

OPTIONS

-d

ディレクトリであれば 0 を返します

-e

存在すれば 0 を返します

-f

通常ファイルであれば 0 を返します

-h

シンボリックリンクであれば 0 を返します

-s

通常ファイルで、サイズが 0 より大きければ 0 を返します

-L

通常ファイルで、実ファイルが同一ホストに格納されていれば 0 を返します

-R

通常ファイルで、実ファイルが同一ホストに格納されていなければ 0 を返します

-?

引数オプションを表示します．
