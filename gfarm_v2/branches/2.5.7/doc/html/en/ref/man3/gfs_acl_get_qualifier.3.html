gfs_acl_get_qualifier

Name

gfs_acl_get_qualifier -- get the qualifier from a Gfarm ACL entry

Synopsis

#include <gfarm/gfarm.h>

gfarm_error_t gfs_acl_get_qualifier(gfarm_acl_entry_t entry_d, char **qualifier_p);

INPUT ARGUMENTS

entry_d

An ACL entry.

qualifier_p

A pointer of the qualifier. Do not free() this *qualifier_p.

RETURN VALUES

GFARM_ERR_NO_ERROR

The function terminated successfully.

Others

An error except the above occurred. The reason is shown by gfarm_error_string(3).
