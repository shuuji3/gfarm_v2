Name

gfmv — Gfarm ファイルを名称変更，移動する

Synopsis

gfmv [options] src dest-file

gfmv [options] src... dest-dir

DESCRIPTION

最初のコマンド形式の場合， Gfarm ファイルをsrcから dest-fileに名称変更します．

2つ目のコマンド形式の場合， srcで指定された Gfarm ファイルを， 既存のディレクトリである dest-dirの下に移動します．

OPTIONS

-f

書き込み権限のないファイルを置き換える場合に，確認を求めないようにします．

-i

ファイルを置き換える場合に，確認を求めるようにします．

-?

引数オプションを表示します．
