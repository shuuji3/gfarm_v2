Name

gfs_pio_datasync — メモリ上のファイルの内容をディスクに反映させる

Synopsis

#include <gfarm/gfarm.h>
gfarm_error_t gfs_pio_datasync( gf);  
GFS_File gf;
 

DESCRIPTION

書き込み用にオープンしているファイル gfの、メモリ上にあるデータを実際 にディスクに書き込み，入出力の完了を待ってから戻ります．メタデータは 更新しません．別の関数 gfs_pio_syncは，ファイルデータに加えてメタデー タもディスクに書き込みます．

RETURN VALUES

GFARM_ERR_NO_ERROR

正常終了を示します．

GFARM_ERR_NO_MEMORY

メモリ不足のため処理を中止しました．

その他

上記以外のエラーが生じたことを示します．
