Name

gfs_pio_flush — 書き込みをフラッシュする。

Synopsis

#include <gfarm/gfarm.h>
gfarm_error_t gfs_pio_flush( gf);  
GFS_File gf;
 

DESCRIPTION

書き込み用にオープンしているファイル gfの、書き込みバッファに溜まって いるデータを実際にディスクに書き込みます。

RETURN VALUES

GFARM_ERR_NO_ERROR

正常終了を示します。

GFARM_ERR_NO_MEMORY

メモリ不足のため処理を中止しました。

その他

上記以外のエラーが生じたことを示します。
