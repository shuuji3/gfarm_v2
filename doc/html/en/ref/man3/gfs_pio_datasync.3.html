Name

gfs_pio_datasync — output a file's data on memroy to disk

Synopsis

#include <gfarm/gfarm.h>
gfarm_error_t gfs_pio_datasync( gf);  
GFS_File gf;
 

DESCRIPTION

The function gfs_pio_datasync flushes data on memory of the gf, and waits for the I/O to complete before returning. It does not update metadtata stat information, while another function gfs_pio_sync updates the metadata in addition to the contents.

RETURN VALUES

GFARM_ERR_NO_ERROR

The function terminated successfully.

GFARM_ERR_NO_MEMORY

Insufficient memory was available.

Others

An error except the above occurred. The reason is shown by gfarm_error_string(3).
