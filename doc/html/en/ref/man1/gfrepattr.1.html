Name

gfrepattr — Manipulate information for replication in Gfarm file system

Synopsis

gfrepattr [ -s | -r ] [ -c | -m ] [-h] ARG [ARG...]

gfrepattr -s [ -c | -m ] [-h] path attr

gfrepattr -r [-h] path [path...]

gfrepattr [-h] path [path...]

DESCRIPTION

gfrepattr manipulates replication attributes of a specified file or a directory. Replication attributes is used by the Replication-Attribute-based File Replication. A replication attribute consists of a hostgroup name and a maximum number of file replica that can create in an hostgroup.

Default action displays all replication attributes about specified files or directories.

OPTIONS

-s

Set replication attributes for a specified file or directories.

-s option needs an attr as replication attributes.

an attr is represented by the following BNF:


        attr := | anattr ',' attr
        anattr := string ':' integer
      

e.g.)


        'group0:2'
        'group0:1,group1:3'
      

-r

Removes any replicaton attributes from a specified file or a directory.

-c

Fails if replication attributes are already specified. This option is used with the -s option.

-m

Fails if replication attributes aren't specified yet. This option is used with the -s option.

-h

Processes symbolic link instead of any referenced files.
