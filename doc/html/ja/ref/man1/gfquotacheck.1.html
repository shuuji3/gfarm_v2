gfquotacheck

Name

gfquotacheck -- 現在の使用量管理表を構築し、quota チェックを有効にする

Synopsis

gfquotacheck [オプション]

DESCRIPTION

gfquotacheck は、すべてのユーザのファイル数と ファイルサイズを数えなおします。 そして、まだ有効になっていないユーザやグループに対する quota によるファ イル制限を有効にします。

このコマンドは gfarmadm グループに所属している管理者が実行できます。

OPTIONS

-P NAME

指定したパス名により特定のメタデータサーバを指定します。

-?

引数オプションを表示します。

gfedquota(1), gfquota(1), gfusage(1)
