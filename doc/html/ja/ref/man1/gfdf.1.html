gfdf

Name

gfdf -- ディスク空き容量取得

Synopsis

gfdf [-a] [-n] [-P パス名] [-D ドメイン名]

DESCRIPTION

gfdf は、Gfarmファイルシステムの ディスク空き容量を表示します。

OPTIONS

-a

Gfarmファイルシステム全体のディスク空き容量を表示します。

-n

利用可能なファイルシステムノードのホスト名を表示します。

-P パス名

指定したパス名により特定のメタデータサーバを指定します． このオプションは -a オプションと同時に利用すると無視されます．

-D ドメイン名

指定したドメイン名に含まれるファイルシステムノードの状態を表示します。

-?

引数オプションを表示します。

SEE ALSO

gfhost(1), gfarm2.conf(5)
