Name

gfs_pio_seek — 指定した位置にシークする

Synopsis

#include <gfarm/gfarm.h>
gfarm_error_t gfs_pio_seek( gf,  
  offset,  
  whence,  
  result);  
GFS_File gf;
gfarm_off_t offset;
int whence;
gfarm_off_t * result;
 

DESCRIPTION

ファイル gf を、 whence と offset で指定した位置にシークします。 引数 offset は、 whence が GFARM_SEEK_SET の場合にはファイル先頭から、 GFARM_SEEK_CUR の場合には現在のシーク位置から、 GFARM_SEEK_END の場合にはファイル末尾からのオフセットです。

引数 result が NULL でない場合には、移動後の位置を result の指す先に返します。

RETURN VALUES

GFARM_ERR_NO_ERROR

正常終了を示します。

GFARM_ERR_NO_MEMORY

メモリ不足のため処理を中止しました。

GFARM_ERR_INVALID_ARGUMENT

指定されたwhenceとoffsetが不正な値です。

その他

上記以外のエラーが生じたことを示します。 gfarm_error_string(3) によりエラーの原因を文字列に変換することができます。
